# std::ranges::subrange<I,S,K>::end

```cpp
constexpr S end() const;  // (since C++20)
```

Returns the sentinel indicating the end of the `subrange`.

### Parameters

(none)

### Return value

A sentinel copy-constructed from the stored sentinel.
