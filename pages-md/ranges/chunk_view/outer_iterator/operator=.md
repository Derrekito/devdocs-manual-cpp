# std::ranges::chunk_view<V>::*outer-iterator*::operator=

```cpp
/*outer-iterator*/& operator=( /*outer-iterator*/&& other ) = default;  // (since C++23)
```

Assigns another iterator. This move assignment operator is generated by the
compiler.

Moves the underlying pointer `other.parent_` to `parent_`.

### Parameters

- **other** — an iterator

### See also

- **(constructor) (C++23)** — constructs an iterator (public member function)
