# std::num_get<CharT,InputIt>::num_get

```cpp
explicit num_get( std::size_t refs = 0 );
```

Creates a `std::num_get` facet and forwards the starting reference count `refs`
to the base class constructor, `locale::facet::facet()`.

### Parameters

- **refs** — starting reference count
