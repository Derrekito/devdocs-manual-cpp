# std::money_get<CharT,InputIt>::money_get

```cpp
explicit money_get( std::size_t refs = 0 );
```

Creates a `std::money_get` facet and forwards the starting reference count
`refs` to the base class constructor, `locale::facet::facet()`.

### Parameters

- **refs** — starting reference count
