# std::stacktrace_entry::operator=

```cpp
constexpr stacktrace_entry& operator=( const stacktrace_entry& other ) noexcept;  // (since C++23)
```

Copy assignment operator. Replaces the contents of `*this` with those of
`other`.

### Parameters

- **other** — another `stacktrace_entry` to assign from

### Return value

`*this`.
