# std::basic_syncbuf<CharT,Traits,Allocator>::swap

```cpp
void swap( basic_syncbuf& other );
```

Exchanges the contents of the sync stream buffer with those of `other`.

### Parameters

- **other** — sync stream buffer to exchange contents with

(none)
