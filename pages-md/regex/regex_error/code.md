# std::regex_error::code

```cpp
std::regex_constants::error_type code() const;  // (since C++11)
```

Returns the `std::regex_constants::error_type` that was passed to the
`std::regex_error` constructor.

### See also

- **error_type (C++11)** — describes different types of matching errors
  (typedef)
