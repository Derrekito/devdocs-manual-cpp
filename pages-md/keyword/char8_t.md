# C++ keyword: char8_t (since C++20)

### Usage

- `char8_t` type: as the declaration of the type (since C++20)

### Notes

  Feature-test macro | Value | Std | Feature
  `__cpp_char8_t` | 201811L | (C++20) | `char8_t`
  202207L | (C++20) (DR) | `char8_t` compatibility and portability fix (allow
      initialization of (`unsigned`) `char` arrays from UTF-8 string literals)
