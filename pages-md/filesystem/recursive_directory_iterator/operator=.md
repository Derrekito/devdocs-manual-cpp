# std::filesystem::recursive_directory_iterator::operator=

```cpp
recursive_directory_iterator& operator=( const recursive_directory_iterator& other ) = default;  // (since C++17)
recursive_directory_iterator& operator=( recursive_directory_iterator&& other ) = default;  // (since C++17)
```

Assigns a recursive directory iterator.

### Parameters

- **other** — another directory iterator to assign

### Return value

`*this`

### Exceptions

May throw implementation-defined exceptions.
