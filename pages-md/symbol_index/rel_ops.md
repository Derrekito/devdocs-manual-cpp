# std::rel_ops Symbol Index

There are no identifiers available from the standard library in the namespace
`std::rel_ops`.

The namespace exists for automatic generation of comparison operators, and is
deprecated in C++20 because of the three-way comparison operator.

### See also

- **operator!=operator>operator<=operator>= (deprecated in C++20)** —
  automatically generates comparison operators based on user-defined
  `operator==` and `operator<` (function template)
