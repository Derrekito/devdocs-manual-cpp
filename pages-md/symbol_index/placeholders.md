# std::placeholders Symbol Index

This page tries to list all the symbols that are available from the standard
library in the namespace `std::placeholders`. The symbols are written as
follows:

- Function names with `()`.
- Templates with `<>`.

## `_`

### _ (underscore)

`_1, _2, ..., _N` (since C++11)
