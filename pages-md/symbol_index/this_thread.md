# std::this_thread Symbol Index

This page tries to list all the symbols that are available from the standard
library (concurrency support library) in the namespace `std::this_thread`. The
symbols are written as follows:

- Function names with `()`.
- Templates with `<>`.

## `G S Y`

### G

`get_id()` (since C++11)

### S

`sleep_for<>()` (since C++11)
`sleep_until<>()` (since C++11)

### Y

`yield()` (since C++11)
