# std::independent_bits_engine<Engine,W,UIntType>::min

```cpp
static constexpr result_type min();  // (since C++11)
```

Returns the minimum value potentially generated by the engine adaptor. This
value is equal to 0u.

### Parameters

(none)

### Return value

The minimum potentially generated value.

### Complexity

Constant.

- **max [static] (C++11)** — gets the largest possible value in the output range
