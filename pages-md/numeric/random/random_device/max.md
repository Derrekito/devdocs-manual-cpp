# std::random_device::max

```cpp
static constexpr result_type max();  // (since C++11)
```

Returns the maximum value potentially generated by the random-number engine.
This value is equal to `std::numeric_limits<unsigned int>::max()`.

### Parameters

(none)

### Return value

The maximum potentially generated value.

### Complexity

Constant.
