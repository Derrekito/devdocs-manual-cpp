# std::uniform_int_distribution<IntType>::max

```cpp
result_type max() const;  // (since C++11)
```

Returns the maximum value potentially generated by the distribution.

### Parameters

(none)

### Return value

The maximum value potentially generated by the distribution.

### Complexity

Constant.
