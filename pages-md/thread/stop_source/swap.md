# std::stop_source::swap

```cpp
void swap( std::stop_source& other ) noexcept;  // (since C++20)
```

Exchanges the stop-state of `*this` and `other`.

### Parameters

- **other** — `stop_source` to exchange the contents with

### Return value

(none)
